HTML PORTFOLIO PROJECT

A portfolio was needed, thus one was created!

I had originally planned on using a website like Wix or Squarespace to create my portfolio. I never liked how restricting and cookie-cutter website builders were, but I had no idea how to make my own website. That was until I took a class on HTML, it was only a single course but it taught me everything I needed to start building that skillset.

YOU ARE LOOKING AT MY FIRST HTML WEBSITE

PROCESS WORK

I started by creating a mock-up of this website in Figma.

I really wanted my personality to shine through and create a memorable portfolio. This meant being ambitious for my first time using HTML, including adding interactions with JavaScript, transitions, and custom assets! All of which had to be researched outside of my course.

I hope the work put into this project has spoken for itself as you navigate this website!
